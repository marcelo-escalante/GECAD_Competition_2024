In this years' competition OpenDSS is needed!

Here are the steps to install it:

1) Go to: https://sourceforge.net/projects/electricdss/
2) Click on the Download button (the sofware is trustworthy don't worry!)
3) Proceed as a normal installation by accepting the agreement
4) Fill the user name and organization fields and install!
5) After the installation you should be albe to run the competition codes
